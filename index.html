About Project

Smart Kitchen AI is an intelligent virtual assistant developed using IBM watsonx Orchestrate. It helps users discover recipes using available ingredients, suggests healthy meal plans, provides food storage guidance, recommends ingredient substitutes, and reduces food waste using Retrieval-Augmented Generation (RAG) and AI.
