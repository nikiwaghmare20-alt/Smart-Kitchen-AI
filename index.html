Smart Kitchen AI

IBM watsonx Orchestrate Project

About Project

Smart Kitchen AI is an intelligent virtual assistant developed using IBM watsonx Orchestrate. It helps users discover recipes, manage ingredients, provide food storage tips, suggest healthy meals, and reduce food waste using AI.
